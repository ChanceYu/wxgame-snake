# 微信贪吃蛇小游戏

一个基于微信小游戏平台开发的经典贪吃蛇游戏，采用像素风格设计。

## ✨ 游戏特点

- 🐍 逼真的蛇形象（绿色条纹、动态舌头、鳞片纹理）
- 🎨 像素风格设计
- 📱 虚拟摇杆控制
- 🏆 分数系统与速度递增

## 🎮 操作说明

- **移动**：使用右下角虚拟摇杆控制方向
- **开始/暂停**：点击屏幕上方区域

## 📦 项目结构

```
wxgame-snake/
├── game.js               # 游戏入口
├── game.json             # 游戏配置
├── js/
│   ├── main.js           # 主逻辑
│   ├── base/             # 工具类
│   ├── libs/             # 适配器
│   ├── npc/              # 游戏对象（蛇、食物、控制器）
│   └── runtime/          # 背景渲染
└── README.md
```

## 🚀 快速开始

1. 使用微信开发者工具打开项目
2. 选择"小游戏"项目类型
3. 点击"编译"运行

## 🎨 视觉风格

- **背景**：深灰色网格
- **小蛇**：绿色条纹，带眼睛、舌头和鳞片
- **食物**：红色苹果
- **界面**：白色文字，右下角控制器
